# Intro
This is my repository that holds Markdown files of my user guides to be generated using MkDocs. 

## Deploy Instructions
1. Install [MkDocs](https://www.mkdocs.org/).
2. Download/Clone this repo.
3. Navigate to the **user-guides-mkdocs** folder.
4. Run the command **mkdocs build**.
5. Upload the generated **site** folder to your hosting provider.
